OutsideIn

Description:

In this game, users can place a variety of different structure blocks together to form larger unique structures. The game provides
a lot of tools to help the user build, such as individual block rotation, entire structure rotations, and block grabbing functionality.
When the user is happy with their build, they can switch the game to play/physics mode, where they throw their structures around and
interact special functional blocks. One such functional block in this game is the rocket booster. This block will apply force to any
structure it is attached to, and can even allow it to fly if the there are enough boosters to overcome gravity. Another functional block
is the seat. This block allows to player to sit on the structure they build and even fly off with it if it has rocket boosters attached.
